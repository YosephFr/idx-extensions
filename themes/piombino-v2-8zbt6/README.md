# Piombino Store Theme V2 8zbt6

Bundle de ejemplo para `idx-developer` inspirado en la web real de El Piombino y adaptado al flujo de ecommerce de `idx-store`.

Incluye:

- logo horizontal real para header
- logo vertical real para footer
- 4 fotos hero reales del sitio de referencia
- layout inmersivo para home, catalogo, producto, carrito y checkout
- footer con credito a Enigma Developers SAC

Comprime esta carpeta y subela como theme store desde el modulo Developer.
