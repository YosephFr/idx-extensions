# Extensions Overview

## Scope

- Public extension repo ownership.

## Source Files

- `docs/`
- `plugins/`
- `themes/`

## Ownership

`idx-extensions` owns:

- plugin development guide
- store theme guide
- booking theme guide
- theme contracts/examples
- plugin examples
- theme snapshots

Runtime enforcement belongs to `idx-engine`; runtime rendering belongs to
`idx-store`, `idx-booking` and admin/developer surfaces in `idx-front`.

## Update Triggers

- New plugin/theme contract.
- New slot.
- New example bundle.
- Store/booking runtime support change.
