# Extensions Overview

## Scope

- Public extension repo ownership.

## Source Files

- `docs/`
- `plugins/`
- `themes/`

## Ownership

`idx-extensions` owns:

- plugin development guide
- store theme guide
- booking theme guide
- theme contracts/examples
- plugin examples
- theme snapshots

## Update Triggers

- New plugin/theme contract.
- New slot.
- New example bundle.
